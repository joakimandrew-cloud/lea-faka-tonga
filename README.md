# Lea Faka-Tonga Interactive Sentence Builder

Interactive web app for learning Tongan sentence structures.

## Setup

```
npm install
npm run dev
```

## Modes

- **Build (Mode 1)** - Select words from dropdowns to build Tongan sentences. See the English translation update as you select.
- **Challenge (Mode 2)** - Given an English sentence, select the correct Tongan words to match it.

## Fonts

Place `CharisSIL-Regular.woff2` in `public/fonts/` for proper Tongan text rendering. Falls back to Georgia/serif if not present.
